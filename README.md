## project-profile-interactive

### 하위 항목들에 대해 소개와 함께 인터렉티브한 웹페이지를 제공합니다. 

#### "Carrer" - 회사에서 진행한 프로젝트

#### "Group" - 다른 개발자들과 진행한 프로젝트

#### "Personal" - 개인적으로 진행한 프로젝트

#### "Skill" - 개발 스킬

### skill
- nextJs 14 (app router)
- typeScript
- react
  
### etc
- Lucid Icon (아이콘 적용)
